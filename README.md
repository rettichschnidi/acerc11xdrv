About acerc11xdrv
=================
This project provides a Linux driver for the Acer C110 pico projector.

USB id:

* 1de1:1101 (c110, when in mass storage mode)
* 1de1:c101 (c110, when in projector mode)

Requirements
============

* libusb-1.0
* libjpeg
* boost
* CMake 2.8
* Doxygen (optional, to generate the documentation)
